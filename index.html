php-build

php-build builds multiple PHP versions from source, and this fully automatic.

Who uses this?

  • Travis CI
  • Maybe you?

Install

% git clone git://github.com/php-build/php-build
% cd php-build
% sudo ./install.sh

Usage

Synopsis

php-build --help
php-build --definitions
php-build [-i <environment_or_inifile>] <definition> <prefix>

Before you can build PHP you will need all necessary build dependencies.

If you happen to use Ubuntu then this is done with one command:

% sudo apt-get build-dep php5-cli

Then run this to install PHP 5.5.9 in development configuration:

% php-build -i development 5.5.9 ~/local/php/5.5.9

To get a list of all supported PHP versions pass the --definitions flag:

% php-build --definitions

By default PHP versions come only with Pyrus (the new Pear installer) preinstalled. Pass the --pear flag to enable the legacy Pear installer alongside of Pyrus for enhanced compatibility with legacy Pear packages:

% php-build -i development --pear 5.5.9 ~/local/php/5.5.9
